# Mad Minute - 수학 계산 문제 앱 요구사항 명세

## 개요

3분 안에 50개의 곱셈 문제를 푸는 수학 연습 앱.

---

## 기능 요구사항

### 1. 시작 화면

- **Start 버튼**이 중앙에 표시됨
- 버튼 클릭 시 타이머 시작과 동시에 문제 화면으로 전환

### 2. 타이머

- 화면 상단에 **3:00**부터 카운트다운
- 시간이 0이 되면 → **자동 제출** (현재 입력 내용으로 채점)

### 3. 문제 영역

- **50개의 곱셈 문제**가 한 화면에 모두 표시
- 각 문제 형식:
  ```
  (0~12 사이 난수) × (0~12 사이 난수) × (0~12 사이 난수) = [빈칸 입력]
  ```
- 숫자는 Start 클릭 시 랜덤 생성, 제출 전까지 변하지 않음
- 정답 입력 칸은 숫자만 입력 가능

### 4. 제출 (Submit)

- **Submit 버튼**이 문제 목록 맨 하단에 위치
- 클릭 시 (또는 타이머 만료 시 자동 실행):
  - 맞은 문제: **녹색** 표시
  - 틀린 문제: **빨간색** 표시
  - 미입력 칸은 틀린 것으로 처리
  - 화면 상단에 **맞은 개수 / 50** 점수 표시
  - **Reset 버튼** 표시 → 클릭 시 처음 Start 화면으로 돌아감

---

## 레이아웃

- 문제는 **5열 × 10행 그리드**로 배치
- 스크롤 가능 (50문제 전체 표시)

## 비기능 요구사항

- 단일 페이지 웹 앱 (별도 서버 없이 브라우저에서 실행)
- 반응형 고려 불필요, 데스크탑 기준
